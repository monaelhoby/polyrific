OPEN CATAPULT

Low Code Development

The OpenCatapult project is an open source low code development platform that runs on your own resources and not only accomplishes the devOps tasks common in today's IT departments, but also generates custom code libraries for each project in your backlog in a manner that fits your specific implementation needs.

We call OpenCatapult the digital developer because it is like adding a new developer to your team. Best of all, it is open, free, and fully customizable.

You can read more about it at the dedicated openCatapult Website.

  • Open source
  • Free
  • Fully customizable
